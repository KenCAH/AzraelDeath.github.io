BZOJ1475 方格取数 <黑白染色+最小割>

  • BZOJ
  • 图论
  • 最小割
  • 网络流

方格取数
Time Limit: 5 Sec Memory Limit: 64 MB

Description
在一个n*n的方格里，每个格子里都有一个正整数。从中取出若干数，使得任意两个取出的数所在格子没有公共边，且取出的数的总和尽量大。

Input
第一行一个数n；（n<=30） 接下来n行每行n个数描述一个方阵
Output
仅一个数，即最大和

Sample Input
2
1 2
3 5
Sample Output
6

标签：黑白染色+最小割

将网格图黑白染色，同色格子间不会冲突。
建模：S->黑格 – val，白格->T – val，黑格->白格 – INF
ans = 总权值-最小割
